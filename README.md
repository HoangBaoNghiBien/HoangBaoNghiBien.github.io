## A. set-up

1. Install the dependencies

   ```sh
   npm install or yarn
   ```

2. Start the development server

   ```sh
   yarn run start
   ```

## B. build and run for production

1. Generate a full static production build

   ```sh
   npm run build
   ```

## C. deploy

   ```sh
   export NODE_OPTIONS=--openssl-legacy-provider
   ```

   ```sh
   npm run deploy
   ```
